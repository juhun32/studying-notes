Ajax(Asynchronous Javascript And XML)

Ajax는 개발자의 꿈이다, 왜냐하면...

  • 페이지가 로드된 후에 웹서버에서 데이터를 읽어들인다.
  • 페이지를 리로드 하지 않고 페이지를 업데이트 한다.
  • 백그라운드에서 웹 서버에 데이터를 보낸다.

Ajax의 정의

  • Asynchronous JavaScript And XML:비동기 자바스크립트 & XML
  • not a programming language: 일종의 오브젝트
    • 사용법:
    • A browser built-in XMLHttpRequest object (to request data from a web server): 브라우저 빌트인 오브젝트(XMLHttpRequest)를 사용해서 웹서버에서 데이터를 읽어들인다.
    • JavaScript and HTML DOM (to display or use the data): 읽어온 데이터를 DOM과 Javascript를 사용해서 보여준다.
